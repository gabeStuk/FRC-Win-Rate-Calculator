# Usage:
 - Clone the repo and run run.bat with `cmd /k run.bat` or with `py frcwr.py` if you've already got pip up to date.
 - You can also run this program from the actions tab by triggering the [Run Script](https://github.com/gabeStuk/FRC-Win-Rate-Calculator/actions/workflows/run.yml) action.
# Reqs:
 - requires [python](https://www.python.org/downloads) (tested and verified for 3.12)

# Note:
 - if not on windows, the `py` shorthand won't work on other OS's, but it's just running a python file after varifying pip is updated properly (so probably `python` or `python3`)
